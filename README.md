# Futbol Femení

## Requisits del sistema per a la IA
Per utilitzar la funcionalitat de generació automàtica de descripcions d'estadis, és necessari descarregar el model `tinyllama:1.1b` dins del contenidor `ollama`.

Executa la següent comanda al teu terminal:
```bash
docker exec ollama ollama pull tinyllama:1.1b
```
